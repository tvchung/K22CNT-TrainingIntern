3. I/O 

4. Các cấu trúc điều khiển (3 loại)
4.1: Lựa chọn / rẽ nhánh
4.1.1: if (4 dạng if)
    a) if đơn
        if(expression-condition){
            statements;
        }
    b) if đẩy đủ 
        if(expression-condition){
            statements - true;
        }else{
            statements - false;
        }
    c) if bậc thang:
        if(expr1){
            statements - expr1=true; 
        }else if(expr2){
            statements - expr2=true; 
        }....

    d) if nested
        if(expr1){
            statements ;
            if(expr2){...}
        }else{

        }
4.1.2: switch
    switch(expression){
        case value1:
            statements;
            break;
        case value2:
            statements;
            break;
            ....

        [default:
            statements;
            break;
        ]
    }

4.2: Loop (Cấu trúc lặp)
4.2.1: Lặp for (4 dạng )
    a) for đầy đủ:
        for(initial ; condition ; step){
            statement block;
        } 
    b) for khuyết:
    *)   for( ; condition ; step){
            statement block;
        } 
    *)  for( initial;  ; step){
            statement block;
        } 
    *)  for( initial; condition ; ){
            statement block;
        } 
    c) for khuyết toàn bộ
        for(; ;){
            statement block;
        }
    d) for nobody
        for(initial, statement ; condition ; step, statement){}
4.2.2: lặp while
    while(condition){
        statement block;
    }
4.2.3: do... while 

    do{
        statement block;
    }while(condition);


4.2.4: for collectioon
    foreach(...)
    for(...)
    forEach(...)
    ....

4.3: Lệnh nhảy
    break;
    continue:
    return; 
    ....
